# Lab Objectives

## Primary Objectives

1. **Build a realistic enterprise environment**
   - Windows Server + Linux integration
   - Multi-zone network segmentation
   - Production-like services

2. **Demonstrate infrastructure engineering skills**
   - Active Directory design and deployment
   - DNS/DHCP configuration
   - File services (DFS, FSRM, Samba)
   - Web services (IIS, Nginx)
   - Monitoring (Zabbix)
   - ITSM (GLPI)

3. **Implement security best practices**
   - Network segmentation
   - DMZ isolation
   - Firewall rules
   - IDS/IPS monitoring
   - Least privilege access

4. **Validate with offensive security testing**
   - Kali Linux attack simulation
   - Realistic attack scenarios
   - Detection validation
   - Incident response practice

## Secondary Objectives

- Document everything (GitHub + LinkedIn)
- Build in public (daily posts)
- Create portfolio content
- Demonstrate problem-solving (troubleshooting)
- Show infrastructure automation (Docker)
